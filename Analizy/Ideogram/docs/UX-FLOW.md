# UX-FLOW — Przepływy Ideogram

## A. Generowanie
1. Pole promptu (tekst) + opcjonalny upload obrazu do edycji.
2. Konfiguracja przed generacją:
   - **Model**: Auto (AI wybiera) | 4.0 latest (2K, best quality) | 3.0 | Legacy | Custom.
   - **Liczba obrazów**: np. 4.
   - **Proporcje**: portret/pejzaż, ~20 wartości (1:3 … 3:1), suwak wymiaru (px), Custom (premium).
   - **Rendering/Speed**: Turbo/Auto/Fast (wpływa na koszt/kredyty).
3. Wysłanie → generacja partii obrazów → wyniki w siatce.
4. Z wyniku: Like/Save do biblioteki, Remix, Edit, Upscale, Remove BG, Layerize.

## B. Explore (społeczność)
1. Masonry-feed obrazów społeczności, licznik polubień na kafelku.
2. Kategorie: All, Poster, T-shirt, Logo, Marketing, Print on demand.
3. Search ("Search all…").
4. Klik kafelka → widok szczegółowy:
   - Obraz + nawigacja lewo/prawo między pracami autora.
   - Prompt / Magic prompt (zakładki).
   - Autor + Follow, data.
   - Akcje: Like, Share, Download, "…".
   - Akcje twórcze: Edit, Remix, Upscale, Remove BG, Layerize text.
   - "View additional details": Type, Model, Aspect ratio, Resolution, Rendering, Seed, Created.

## C. Edycja / Studio
- Wejście przez "Edit image" / "Image Studio" lub akcje na obrazie.
- Operacje: edycja (inpaint/region), upscale, usuwanie tła, layerize text (rozdzielenie warstwy tekstu), background remover (AI App).
- Po edycji: kontynuacja w Studio lub download.

## D. Custom Models
- "Create custom model to capture your signature look" — trenowanie modelu na własnym stylu/tożsamości, stosowanym spójnie w kolejnych promptach.

## Zasady UX
- Repro: każdy obraz ma seed + pełne metadane → powtarzalność i remix.
- Niski próg: Auto model + Magic Prompt + Prompt Builder pomagają początkującym.
- Społeczność napędza inspirację (remix cudzych prac jako punkt startu).
- Kredyty/plany widoczne stale (motywacja do Upgrade).
