# COMPETITIVE-ANALYSIS — Ideogram w kontekście naszego produktu

## Mocne strony Ideogram
- Silnik generatywny z wieloma modelami + tryb Auto (dobór modelu).
- Magic Prompt (rozszerzanie promptu) + Prompt Builder — obniża próg.
- Pełne metadane + seed → powtarzalność i remix.
- Warstwa społecznościowa (feed, like, follow, remix) napędza odkrywanie.
- Narzędzia edycyjne: upscale, remove bg, layerize text, edit/inpaint.
- Custom modele (signature look) — personalizacja/monetyzacja.
- Kategorie komercyjne (poster/t-shirt/logo/POD) — nastawienie na zastosowania.

## Luki / czego brakuje względem naszego celu
- Brak "doradcy wnętrza" (co pasuje do MOJEGO pokoju/ściany) — to domena Fy!.
- Brak pipeline produkcji fizycznej (rama/rozmiar/wysyłka) — to domena Mixtiles.
- Generacja jest celem samym w sobie, nie prowadzi do gotowego produktu na ścianie.

## Wniosek
Ideogram wnosi najlepszy "silnik + społeczność + edycja". W naszym produkcie to warstwa tworzenia treści, którą spina advisor (Fy!) i pipeline produktowy (Mixtiles).
