# MERGE-STRATEGY — Ideogram + Mixtiles + Fy! w jeden produkt "2026"

## Rola każdego filaru
- **Fy! (advisor)**: rozumie intencję/wnętrze, dobiera i uzasadnia ("mózg doboru").
- **Ideogram (silnik)**: generuje/edytuje grafikę, remix, custom style, repro (seed) ("ręce twórcze").
- **Mixtiles (produkcja)**: rama/rozmiar/efekt, gotowe layouty, magnetyczne wieszanie, checkout ("fabryka i sklep").

## Zunifikowany przepływ
1. Wejście: quiz LUB brief w języku naturalnym + opcjonalny upload zdjęcia wnętrza (Fy!).
2. Advisor ustala kierunek (paleta, styl, format, układ ściany).
3. Źródła treści zbiegają się:
   - własne zdjęcia (edytor Mixtiles),
   - GENERACJA AI (silnik typu Ideogram: prompt/enhancement/custom model),
   - katalog gotowej sztuki + kurowane layouty (Mixtiles/Fy!).
4. Advisor komponuje ścianę z wygenerowanych/dobranych prac, nadaje nazwę, tłumaczy "dlaczego".
5. Personalizacja fizyczna (rama/rozmiar/efekt) + wizualizacja na ścianie (2.5D → inpainting w zdjęciu wnętrza).
6. Checkout + wysyłka + magnetyczne wieszanie.

## Kluczowe elementy przeniesione z Ideogram
- Prompt enhancement + Prompt Builder (własne).
- Seed + pełne metadane → remix i powtarzalność wariantów ściany.
- Custom modele = "styl marki/klienta" (spójna estetyka całej ściany/serii).
- Similarity search (feed inspiracji + "dobierz podobne do tej pracy").
- Narzędzia: upscale (druk w wysokiej rozdzielczości!), remove bg, layerize text (personalizowane napisy na plakatach).

## Synergia druku
Upscale + wysokie rozdzielczości Ideogram (np. 2K, 1024×3072) idealnie pasują do druku wielkoformatowego Mixtiles — to naturalny most technologiczny.

## Ryzyka
- Koszt GPU generacji + druku — sterować kredytami i cache'em.
- Prawa autorskie/licencje do wygenerowanych prac (jasny model licencyjny dla klienta).
- Moderacja treści przed drukiem i publikacją do feedu.
