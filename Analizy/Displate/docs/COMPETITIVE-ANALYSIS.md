# COMPETITIVE-ANALYSIS — Displate w kontekście naszego produktu

## Mocne strony Displate
- Marketplace twórców (ogromna, różnorodna podaż) + licencje IP.
- Premium materiał (metal) = wyższy AOV, pozycjonowanie kolekcjonerskie.
- Fizyczna innowacja: magnetyczne wieszanie bez narzędzi (łatwość, wymienialność).
- Silne mechaniki fandomowe (breadcrumbs, kategorie, kolekcje).
- Dojrzała monetyzacja: Club (subskrypcja), Limited drops, rabaty ilościowe, sety.
- Compliance i zaufanie (Omnibus, Trustpilot, gwarancje/zwroty).

## Luki / czego brakuje względem naszego celu
- Brak konwersacyjnego doradcy wnętrza (domena Fy!).
- Brak generatywnego AI z promptu/uploadu jako rdzenia tworzenia (domena Ideogram).
- Personalizacja custom jest prosta (upload), bez kompozycji ściany/wizualizacji jak Mixtiles.

## Wniosek
Displate wnosi model biznesowy (marketplace + premium + subskrypcja + fandom) i najlepsze praktyki commerce. W naszym ekosystemie to warstwa "rynku i monetyzacji" nad tworzeniem (Ideogram), doborem (Fy!) i personalizacją/produkcją (Mixtiles).
