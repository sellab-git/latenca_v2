# UX-FLOW — Przepływy Displate

## A. Odkrywanie (katalog)
1. Strona główna: hero z promocją (countdown, kod), kafle kategorii (Gaming, Anime, Verified Creators), sekcje "Browse posters you'll love" (For You / Bestselling / Trending / By Verified Creators).
2. Wyszukiwarka (autouzupełnianie, "Top Searches", "Fandoms").
3. Katalog/wyniki (`/search?q=`, `/browse`): pasek filtrów kategorii (gaming, anime, sport, space, cars, fantasy, movies, maps, music, retro, funny, comics…), "ALL FILTERS", "By Verified Creators".
4. Karty produktów z badge'ami wykończenia (np. TEXTRA) i podglądem.

## B. Strona produktu
1. Galeria: miniatury + render na ścianie + wideo lifestyle; breadcrumbs (fandom → kolekcja → tytuł).
2. Sekcja marki/licencji: "Officially licensed · N Artworks", ocena Trustpilot + liczba recenzji, opis (Read more).
3. Konfigurator:
   - **Select product type**: Matte / Gloss / (Textra).
   - **Choose size**: M (45×32cm) / L / XL.
   - **Add frame**: None / Natural / Graphite / White / Black.
4. Cena: aktualna + przekreślona "Lowest price from last 30 days" (Omnibus), "Sale ends in" countdown, rabat ilościowy ("€X for 2+ Displates -25%").
5. ADD TO CART → koszyk.
6. Sekcje dodatkowe: powiązane produkty, "Recently viewed", USP ("Damage-resistant", "Tool-free magnet mounting", "Gift-ready packaging").

## C. Custom Displates
1. "Custom Displates are here! Upload your own pics" → START CREATING.
2. Upload zdjęcia/grafiki → wybór rozmiaru/wykończenia → podgląd → koszyk.

## D. Limited Editions
- Osobny wjazd (`/limited-editions`): edycje ograniczone/ultra-limited, unikalne wykończenia, komunikat niedostępności po wyprzedaniu ("once they're gone - it's forever").

## E. Displate Club
- Landing subskrypcji: perki (free shipping, early access do dropów, więcej). Wejście z topbara.

## Zasady UX
- Silne kotwiczenie fandomem (breadcrumbs, kategorie, "let your fandom shine").
- Wszechobecne mechaniki pilności (countdown) i wartości (przekreślone ceny, rabaty ilościowe).
- Marketplace: ekspozycja artysty i "Verified Creators" buduje zaufanie i różnorodność.
- Prawnie: widoczna zgodność Omnibus (najniższa cena z 30 dni).
