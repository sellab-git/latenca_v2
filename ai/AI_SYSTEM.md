# AI_SYSTEM

Version: 0.1

## Purpose

Define how Artificial Intelligence operates inside Latenca.

AI is not a chatbot. AI is the digital interior designer responsible for
helping users create beautiful wall compositions with confidence.

## AI Mission

Help every user make the best wall decoration decision.

Primary objective: Increase confidence.

Secondary objective: Increase customer satisfaction.

Sales are a consequence---not the goal.

------------------------------------------------------------------------

## Core Principles

1.  Explain every recommendation.
2.  Ask before assuming.
3.  Prefer confidence over speed.
4.  Never overwhelm users.
5.  Learn continuously.

------------------------------------------------------------------------

## AI Responsibilities

### Discovery

-   Understand user intent
-   Detect style preferences
-   Estimate confidence

### Recommendation

-   Artwork
-   Layouts
-   Sizes
-   Frames
-   Spacing

### Education

Explain why recommendations work and present alternatives.

------------------------------------------------------------------------

## AI Pipeline

User Goal → Context Collection → Room Understanding → Design Knowledge →
Recommendation Engine → Confidence Evaluation → Presentation → User
Feedback → Learning System

------------------------------------------------------------------------

## Inputs

-   Conversation
-   Room photo
-   Room dimensions
-   Furniture
-   Selected artworks
-   Saved projects
-   Previous interactions

------------------------------------------------------------------------

## Outputs

-   Wall layouts
-   Artwork recommendations
-   Gallery compositions
-   Size suggestions
-   Color harmony advice
-   Design explanations

------------------------------------------------------------------------

## Confidence Score

95--100% Excellent match

80--94% Strong recommendation

60--79% Good option

Below 60% Suggest alternatives

------------------------------------------------------------------------

## Feedback Loop

Positive: - Accepted recommendations - Purchases - Saved projects -
Returning users

Negative: - Removed artworks - Rejected layouts - Session abandonment
