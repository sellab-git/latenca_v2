# ROADMAP

Version: 0.1

## Purpose

Define the long-term evolution of Latenca from MVP to a world-class AI
wall design platform.

------------------------------------------------------------------------

# Product Vision Timeline

## Phase 0 --- Foundation (Current)

Goals: - Define product vision - Build documentation - Research users -
Benchmark competitors - Design MVP

Status: In Progress

------------------------------------------------------------------------

## Phase 1 --- MVP

Objective: Validate that users value AI-assisted wall decoration.

Scope: - Mobile-first web app - Artwork catalog - AI Designer
(rule-based) - Wall projects - Checkout - Saved projects

Success Metrics: - First paying customers - Positive qualitative
feedback - AI recommendation acceptance \> 50%

------------------------------------------------------------------------

## Phase 2 --- Product-Market Fit

Goals: - Improve recommendation quality - Faster project creation -
Better onboarding - Analytics - User feedback loop

New Features: - Community inspiration - Better search - Collections -
Personalization

------------------------------------------------------------------------

## Phase 3 --- Learning Platform

Goals: - AI learns from user behaviour - Continuous ranking
improvements - Confidence scoring - Better layout generation

------------------------------------------------------------------------

## Phase 4 --- Professional Platform

Audience: - Interior designers - Airbnb hosts - Offices - Hotels

Features: - Team workspaces - Shared projects - Bulk ordering -
Professional tools

------------------------------------------------------------------------

## Phase 5 --- AI Interior Design Platform

Vision: Latenca evolves beyond wall art.

Potential areas: - Complete room styling - Furniture suggestions -
Lighting advice - Multi-room planning - AR assistance

------------------------------------------------------------------------

# Guiding Principles

-   Validate before scaling.
-   Build trust before monetization.
-   Simplicity over feature count.
-   AI should reduce decisions, not increase them.

------------------------------------------------------------------------

# Success Definition

Latenca becomes the default place people visit when they need help
designing a wall.
