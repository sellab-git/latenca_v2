# WALL_DESIGN_KNOWLEDGE

Version: 0.1

## Purpose

This document is the foundation of Latenca's AI recommendation engine.

It captures professional interior design principles for selecting,
arranging and presenting wall art.

The long-term goal is to transform design knowledge into structured
rules that AI can use when generating recommendations.

------------------------------------------------------------------------

# Core Principles

1.  The wall is the product, not the individual artwork.
2.  Every recommendation must consider the room context.
3.  Composition matters more than the number of artworks.
4.  Empty space is part of the design.
5.  Simplicity usually wins.

------------------------------------------------------------------------

# Room Context

Before recommending artwork, AI should understand:

-   Room type
-   Wall dimensions
-   Ceiling height
-   Furniture placement
-   Viewing distance
-   Existing colors
-   Lighting
-   Interior style

------------------------------------------------------------------------

# Wall Types

-   Above sofa
-   Above bed
-   Dining room
-   Hallway
-   Staircase
-   Office
-   Gallery wall
-   Large empty wall
-   Narrow wall

Each wall type will have dedicated recommendation rules.

------------------------------------------------------------------------

# Artwork Size Rules

General heuristics:

-   Artwork should typically occupy 60--75% of the furniture width
    beneath it.
-   Oversized artwork is preferable to artwork that is too small.
-   Very small artwork should rarely be displayed alone on large walls.

These rules will be refined with research.

------------------------------------------------------------------------

# Gallery Wall Rules

Possible layouts:

-   2-piece diptych
-   3-piece triptych
-   4-piece grid
-   6-piece gallery
-   9-piece gallery
-   Mixed-size gallery

Future versions will define spacing, proportions and placement for each
layout.

------------------------------------------------------------------------

# Spacing Guidelines

Initial heuristics:

-   Consistent spacing creates harmony.
-   Larger compositions require more breathing room.
-   Gallery spacing should remain visually consistent.

Exact measurements will be defined later.

------------------------------------------------------------------------

# Height Guidelines

Default recommendation:

Artwork center should generally align near average eye level, adjusted
for furniture and room function.

Future versions will include room-specific rules.

------------------------------------------------------------------------

# Color Harmony

AI should evaluate:

-   Dominant colors
-   Accent colors
-   Contrast
-   Saturation
-   Temperature
-   Existing room palette

Recommendation goal:

Complement the room rather than dominate it.
